## 🧰 Getting Started

### 💻 Prerequisites

Node.js - _download and install_

```
https://nodejs.org
```

Git - _download and install_

```
https://git-scm.com
```

1. Clone the repo

```sh
git clone https://github.com/vitaliju/bit-html-labirintas.git
```

2. Install nPM packages

```sh
npm i
```

or

```sh
npm install
```

3. Run the server

```sh
npm run dev
```
